ProtectionID 6

Overview

ProtectionID 6 is a tool that scans games using older DRM (2004-2011) and can detect what type of DRM is being used. The program can detect DLLs, DAT files, EXE files and supports CD Lock, Codelok, Laserlock, Safecast, Safedisc, SecuROM, Starforce, VOB Protect CD, CD-Checks and more.

SecuROM Diagnostic Tool (by SecuROM)

Overview

This official tool will diagnose any SecuROM errors you may encounter when trying to run a legitimate SecuROM game, and encounter a failure. It will also instruct you on how to fix the errors, however the fixes are not definite, and the tool is known to fail on occasion.

SecuROM Removal Tool (by SecuROM)

Overview

This official tool will remove SecuROM leftover files after a SecuROM game is removed from a computer. This will NOT crack your game and is not designed to, and is only designed to remove leftover files. You use this tool at your own risk, as unfortunately, there have been known incidents of corruption.
