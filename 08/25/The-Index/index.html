index

Introduction

Welcome to “The Index”, a massive list that encompasses all files dredged up by the LostFileArchives team. It’s sorted in alphabetical order, and files can be downloaded from the links in the name. Select files may not be able to download directly from the website, and must be downloaded via. contact with one of the website administrators so they can copy out the archive. Have fun.

There are two types of storage on the website, cold storage and hot storage. Cold storage is the storage that’s permanently available and frozen on the site, and these are the ones that are always available to download on-demand. Hot storage files are the ones that are archived on completely separate offline hard drives, and must be requested via. contact with one of the site administrators, as the files need to be kept offline and safe in storage, or are simply not frequently required. As mentioned, please contact us through our “Contact” page in this instance, and we will be happy to help.

A-Ray DRM Scanner

Overview

A quick DRM checking tool that scans older games (2005-2009) to detect what type of DRM is being used, be it SafeDisc, SecuROM, Starforce or others.

Download

Download Archive Entry - ARay DRM Scanner

 

DLL Rebaser by Doctor Golova

Overview

A simple to use tool that will allow you to rebase the positions of loaded DLLs in memory, to tamper with programs. Used in the Securom 7 writeup, available on the blog.

Download

Download Archive Entry - DLL Rebaser (Golova)

 

Generic Securom 7/8 by Virusek

Overview

This is a SecuROM workaround created by Virusek in 2008 and shared around the CSRIN forum. It allows a basic bypass of the DRM on most SecuROM 7/8 games.

Download

Download Archive Entry - Generic SecuROM (Virusek)

 

IDA32 v70 - Free Edition

Overview

A decompilation and memory management tool used to change memory values of a debug program, and to dump executable files. Similar in function to OllyDBG, but less advanced.

Download

Download Archive Entry - IDA32

 

Lytro Compatible Communicator/Viewer (PC)

Overview

This program was 100% developed by Miloush, who you can find here: http://optics.miloush.net/lytro/Default.aspx
This is simply a mirror in case the site goes down, which would mean nearly all Lytro resources are lost forever.

This is a program that allows you to communicate wirelessly with your Lytro camera, and also view pictures taken.

Download

Download Archive Entry - LytroCC (PC)

 

Lytro Desktop (PC/MAC)

Overview

The official Lytro Desktop program, by Lytro themselves. This provides basic functionality to all Lytro cameras, and is essentially a requirement for using one.

LostFilesArchive is in no way affiliated with Lytro or Google, and is hosting these for archive purposes.

Download

Lytro Firmware

Overview

The firmware for Lytro first edition cameras, not including the ILLUM models. This is 100% needed for the cameras to work, unless firmware is already installed onto the camera.

LostFilesArchive is in no way affiliated with Lytro or Google, and is hosting these for archive purposes.

Download

Download Archive Entry - Lytro Firmware

 

Lytro PowerTools (PC/MAC)

Overview

The open-sourced power tools that were available for Lytro 3D cameras. The company has since been bought by Google and the products discontinued, so the toolkit has been archived here.

Download

Download Archive Entry - Lytro PT (PC)
Download Archive Entry - Lytro PT (MAC)

 

OEPFinder 1.6 by Human/MiNT

Overview

OEPFinder is a tool for finding the OEP of executable files. It was released in 2007 along numerous other tools made by Human. This is used in the SecuROM 7 writeup.

Download

Download Archive Entry - OEPFinder

 

OllyDBG

Overview

OllyDBG is a program for editing and debugging executable files that are loaded into it. You can perform step-by-step debugging, and also dump edited files using the OllyDump plugin.

Download

Download Archive Entry - OllyDBG

 

ProtectionID 6

Overview

ProtectionID 6 is a tool that scans games using older DRM (2004-2011) and can detect what type of DRM is being used. The program can detect DLLs, DAT files, EXE files and supports CD Lock, Codelok, Laserlock, Safecast, Safedisc, SecuROM, Starforce, VOB Protect CD, CD-Checks and more.

Download

Download Archive Entry - ProtectionID 6.40

 

SecuROM Diagnostic Tool (by SecuROM)

Overview

This official tool will diagnose any SecuROM errors you may encounter when trying to run a legitimate SecuROM game, and encounter a failure. It will also instruct you on how to fix the errors, however the fixes are not definite, and the tool is known to fail on occasion.

Download

Download Archive Entry - SecuROM Removal Tool

 

SecuROM Removal Tool (by SecuROM)

Overview

This official tool will remove SecuROM leftover files after a SecuROM game is removed from a computer. This will NOT crack your game and is not designed to, and is only designed to remove leftover files. You use this tool at your own risk, as unfortunately, there have been known incidents of corruption.

Download

Download Archive Entry - SecuROM Removal Tool

Skylanders: Spyro’s Adventure (PC)

Overview

The toys-to-life game by Activision, published in 2011. This is an archive of the PC edition of the game, with all required SecuROM information included in the copy. This is kept in hot storage.

Download

Ultimate Hooking Engine by Deroko

Overview

Ultimate Hooking Engine is easy to use hooking engine for Win32 APIs. All you need to do is to provide hooking DLL and the engine will perform hooking.

As described by the maker, Deroko. Used in the SecuROM 7 writeup.
