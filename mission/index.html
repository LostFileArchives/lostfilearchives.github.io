Consider this an “About” page, and a statement of the purpose of this site.

As mentioned on the very front page of this website, this archive is dedicated to preserving many old, lost, or plain unavailable programs, files and data to preserve them for anyone to use in the future. This site is run for free, and will always be free and non-profit. Many documents, games, drivers and programs disappear from the web every day from server purges, domain name expirations, and loss from the deep web. This site aims to correct that by giving a safe haven to any and all “missing” files from all over the internet, to be kept in archived storage forever. All files hosted directly on this site are under the correct license and have their license documents included within the archive.

If you wish to contribute to the archives, please e-mail us at lostfilearchives@gmail.com, or contact me via. GitHub at @c272.
